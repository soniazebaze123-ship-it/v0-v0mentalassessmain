## 🚨 COMMON ISSUES & FIXES

| Issue | Cause | Fix |
|-------|-------|-----|
| Blank white page | Missing env vars | Add SUPABASE vars in Vercel |
| 404 errors in console | Missing files | Verify all components exist |
| Database connection error | Supabase not configured | Check API keys, run migrations |
| Assessment won't load | Language context error | Hard refresh (Ctrl+Shift+R) |
| Buttons don't respond | JavaScript error | Check console (F12) |
